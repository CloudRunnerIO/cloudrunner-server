Billing module¶

GET /billing/account/¶

Load basic billing info

Example response:

HTTP/1.1 200 OK
Content-Type: application/json; charset=UTF-8

{
  "billing": {
    "cards": {
      "expire": "11/2015",
      "type": "MasterCard",
      "number": "555555******4444"},
    "plan": {},
    "transactions": []
  }
}
Request Headers:
 
  • X-Cr-User – User name
  • X-Cr-Token – Auth token
Status Codes:
  • 200 OK – no error
PUT /billing/account/¶

Update payment info

Example request:

PUT /billing/account HTTP/1.1
Host: api.cloudrunner.io
Accept: application/json

{
  "cc": {
      "cardholder_name": "John Doe",
      "cvv": "134",
      "number": "4111111111111111",
      "expire_date": "11/21"
    }
}

Example response:

HTTP/1.1 200 OK
Content-Type: application/json; charset=UTF-8

{
  "success": {
    "msg": "Account updated"
  }
}
Request Headers:
 
  • X-Cr-User – User name
  • X-Cr-Token – Auth token
Status Codes:
  • 200 OK – no error
